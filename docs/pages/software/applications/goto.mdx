## Overview

"goto" helps us to force a link in an [App](./app.mdx) to open in a browser.

```mermaid
graph TD
    linkClick["User clicks goto.republik.ch/[path + query]"] --> openBrowser("opens in Browser")
    openBrowser --> parsePathQuery("parses [path + query]")
    parsePathQuery --> foward("fowards to republik.ch/[path + query]")
```
